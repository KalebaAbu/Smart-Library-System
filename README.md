Smart Library System

This is a console‑based library management system written in Python.  
It helps librarians manage books, users, borrowing, returns, and fines

 Features

1. Admin & User roles – Admin can add/remove books, register users, and view all books/users.  
2. Borrow books – Users can borrow up to 3 books. Due date is set 7 days after borrowing.  
3. Return books – If a book is returned late, a fine of $1 per overdue day is added to the user’s account.  
4. Search books – Search by title (case‑insensitive).  
5. Show all books – Books are displayed sorted alphabetically by title.  
6. Overdue warnings – Admin can see which books are overdue and who borrowed them.  
7.Data persistence – All books and users are saved to JSON files (books.json, users.json). When you restart the program, your data is automatically loaded.
